# Mini Event Finder (JavaScript - In-memory)

## Overview
Ready-to-run full-stack project:
- Backend: Node.js + Express (in-memory storage)
- Frontend: React + Vite

## Run backend
```bash
cd backend
npm install
npm run dev
```
Backend will run at http://localhost:4000

## Run frontend
```bash
cd frontend
npm install
npm run dev
```
Frontend will run at http://localhost:5173

## Notes
- Data is stored in memory (it resets when the backend restarts).
- If ports are in use, change them in `backend/src/index.js` and `frontend/vite.config.js`.
